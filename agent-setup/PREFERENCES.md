# Предпочтения

## Рабочий режим

Часовой пояс: (из настроек)
Рабочие часы: (заполнится из разговоров)

## Стиль работы

Автономность: почти автономный. По умолчанию действуй сам по зелёным и жёлтым операциям, не дёргай по мелочам. Периодически сверяйся — кратко показывай, что сделал и что планируешь дальше. Спрашивай только перед красными/необратимыми действиями (удаление, оплата, отправка данных наружу).
Детализация: подробно и пошагово — с примерами и понятной логикой, простым языком без перегруза терминами.
Язык: русский
Инструменты: (заполнится из разговоров) — сейчас основной: таблицы и выгрузки статистики.

## Ограничения

- Слишком сложный, перегруженный терминами язык — нужно объяснять простыми словами.
